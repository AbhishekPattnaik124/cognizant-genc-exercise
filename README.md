# Cognizant GenC Exercises

Welcome to the **Cognizant GenC Exercises** repository! This repository contains a collection of training exercises, projects, and assignments completed as part of the Cognizant GenC training program.

## 📚 Topics Covered

This repository is structured by technology and concept, encompassing a wide range of modern software engineering practices:

- **[Advanced SQL Server Exercises](./Advanced_SQL_Server_Exercises)**: Database management, complex queries, and SQL Server optimizations.
- **[Angular Exercises](./Angular_Exercises)**: Frontend web development using the Angular framework.
- **[Engineering Concepts Exercises](./Engineering_Concepts_Exercises)**: Core software engineering principles, design patterns, and architecture.
- **[Entity Framework Core Exercises](./Entity_Framework_Core_Exercises)**: Object-Relational Mapping (ORM) and data access using EF Core.
- **[Git Exercises](./Git_Exercises)**: Version control workflows, branching strategies, and collaboration.
- **[Microservices Exercises](./Microservices_Exercises)**: Designing, building, and deploying microservices architectures.
- **[NUnit & Moq Exercises](./NUnit_Moq_Exercises)**: Unit testing, mocking, and test isolation using NUnit and Moq in .NET.
- **[React Exercises](./React_Exercises)**: Frontend UI development utilizing the React library.
- **[WebAPI Exercises](./WebAPI_Exercises)**: Building RESTful APIs and backend services using ASP.NET Core.

## 🚀 Getting Started

To explore these exercises on your local machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/AbhishekPattnaik124/cognizant-genc-exercise.git
   ```

2. **Navigate to the specific exercise directory:**
   ```bash
   cd cognizant-genc-exercise
   cd <Exercise_Folder>
   ```

3. **Follow the instructions within each specific exercise folder** (if provided) to build and run the projects. Most .NET projects can be run using the standard `dotnet run` or `dotnet test` CLI commands.

## 🛠️ Prerequisites

Depending on the exercise you are reviewing, you may need the following installed:
- [.NET SDK](https://dotnet.microsoft.com/download) (for C#, WebAPI, EF Core, Microservices, and NUnit)
- [Node.js and npm](https://nodejs.org/) (for Angular and React)
- [SQL Server](https://www.microsoft.com/en-us/sql-server/sql-server-downloads) & [SSMS](https://docs.microsoft.com/en-us/sql/ssms/download-sql-server-management-studio-ssms) (for Database exercises)
- [Git](https://git-scm.com/)
